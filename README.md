# Mini F1 Dashboard (Fan Project)

## Site URL

https://mini-f1-dashboard.vercel.app/

## Development

### Stack used in development

- Typescript
- React
- Tailwind Css

### Tools used in development

- npm
- vite

### Deployment

- Hosted on Vercel with GitHub for continuous integration.

### API

- OpenF1 Api

## Hidden Feature

- Pressing 'Ctrl/Cmd + Shift + F1' triggers hidden experimental feature which will display both [json] data.

## Project description

This is non-commertial fan project featuring F1 drivers data from 2025 Abu Dhabi GP. Data is coming from API listed above (OpenF1 Api), specifically 2 endpoints, using driver numbers to match [json] data and display data on cards as a list. code features fallback data and conditionall chaining to guarantee that this web app won't crash or use 'undefined' data.
